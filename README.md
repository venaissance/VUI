# Steps to Checkout Examples

1. `git clone https://github.com/venaissance/VUI`
1. `yarn install`
1. `yarn doc`

Follow above 3 steps, the example page will show in your `./doc` directory, you can open `doc/index.html` in your browser! 

Have fun!
